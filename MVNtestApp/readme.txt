Maven tuturial:--
1. create one folder >>> mkdir myapp
2. initialize maven >>>> mvn archetype:generate
3. mvn compile
4. mvn package >>> tp convert it into jar file and keep it in target
5. java -cp target/MVNtestApp-1.0-SNAPSHOT.jar org.anand.jha.App >> to run App
